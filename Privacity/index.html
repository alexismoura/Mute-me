Política de Privacidade – MuteMe

Última atualização: 07 de maio de 2025

O aplicativo MuteMe foi desenvolvido com o objetivo de oferecer aos usuários uma ferramenta prática para bloquear chamadas e mensagens SMS indesejadas.

Coleta de Dados

O MuteMe não coleta, armazena ou compartilha nenhum dado pessoal dos usuários.

As permissões solicitadas (acesso a chamadas e SMS) são utilizadas exclusivamente para o funcionamento dos recursos de bloqueio de chamadas e mensagens, sem qualquer envio de informações para servidores externos.

Permissões do App

  • Acesso a chamadas: utilizado apenas para identificar e bloquear chamadas de números não autorizados.
  • Acesso a SMS: utilizado apenas para identificar e bloquear mensagens indesejadas.
  • Contatos (opcional): usado para comparar com a lista branca de contatos, caso o modo de bloqueio completo esteja ativado.

Compartilhamento de Dados

O MuteMe não compartilha dados com terceiros, nem utiliza serviços de terceiros para coleta de informações.

Publicidade

No momento, o aplicativo não exibe anúncios e não utiliza o ID de publicidade do dispositivo.

Alterações nesta Política

Podemos atualizar esta política periodicamente. A versão mais recente estará sempre disponível nesta página.
