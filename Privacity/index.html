Política de Privacidade – MuteMe

Última atualização: 14 de maio de 2025

O MuteMe é um aplicativo criado para bloquear chamadas e mensagens SMS indesejadas. Prezamos pela sua privacidade e protegemos suas informações pessoais de forma transparente.

1. Coleta e uso de dados

O MuteMe não coleta, armazena ou compartilha dados pessoais identificáveis. Todas as permissões são utilizadas exclusivamente para garantir o funcionamento das funcionalidades de bloqueio local no seu dispositivo.

2. Permissões utilizadas

As seguintes permissões são requeridas pelo app:

  • Chamadas telefônicas: Para identificar e bloquear ligações indesejadas.
  • Mensagens SMS: Para bloquear mensagens de texto não autorizadas.
  • Contatos (opcional): Para permitir exceções ao bloqueio com base na sua lista de contatos.
  • Notificações: Para alertá-lo sobre chamadas ou mensagens bloqueadas.

Essas permissões são requeridas exclusivamente para a operação local do app e não envolvem envio ou processamento externo de dados.

3. Compartilhamento de dados

O MuteMe não compartilha seus dados com terceiros. O app também não envia nenhuma informação para servidores externos ou redes de terceiros.

4. Publicidade

O MuteMe pode vir a utilizar o serviço Google AdMob para exibir anúncios. Caso isso ocorra, o app poderá solicitar o ID de publicidade do seu dispositivo, conforme exigido pelas diretrizes do Google.

Você será informado e poderá consentir com o uso dessas permissões, conforme requerido pela lei de proteção de dados aplicável.

5. Segurança

O app não armazena dados do usuário em nuvem. Todas as operações de bloqueio ocorrem no próprio dispositivo.

6. Alterações nesta política

Poderemos atualizar esta política para refletir mudanças no aplicativo ou nas exigências legais. Recomendamos revisar esta página periodicamente.

7. Contato

Em caso de dúvidas sobre esta política, entre em contato pelo e-mail: dragsystem.com@gmail.com
